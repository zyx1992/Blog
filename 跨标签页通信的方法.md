# 跨标签页通信的方法

## api

 - `localStorage`
 - `sessionStorage`
 - `cookie`
 - `postMessage`
 - `BroadcaseChannel`
 - `window.name`
 - `document.domain`
  
## 特性
  
 - 通信
     - 同源/不同源
     - `window.open`打开/随意方式打开
 - 存储
     - 刷新当前页面,丢失内容/不丢失内容
     - 多个窗口共享/不共享数据
     
 - 存储级别
     - 当前窗口刷新内容丢失
     - 当前窗口刷新内容不丢失
     - 关闭当前窗口内容丢失
     - 关闭当前浏览器内容不丢失
